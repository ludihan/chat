# chat

this is a simple chat application made with go and typescript

the `frontend` directory has all the code related to the web frontend
(made with react and vite)

the `backend` directory has all the code related to the web backend
(made with go)

the `mobile` directory has all the code related to the mobile app
(made with react native and expo)

each directory has a `README.md` with instructions on how to build each part of the application
